# Canon

Biblioteca canonica del Diari d'en Reiv.

Carrega nomes les fitxes necessaries per a la tasca concreta. El canon no s'ha de carregar sencer si l'acte nomes menciona una part del mon.

## Carpetes

- `personatges/`: fitxes visuals i identitaries de personatges recurrents.
- `naus/`: fitxes visuals de vaixells i naus recurrents.
- `llocs/`: espais recurrents, quan calgui formalitzar-los.
- `objectes/`: artefactes, armes, reliquies o objectes recurrents, quan calgui formalitzar-los.
